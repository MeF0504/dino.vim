# Dino.vim

vim plugin like chrome://dino

<img src=images/dino.gif width="70%">

## Requirements

- Nothing (Maybe).

## Installation

If you use dein,
```vim
call dein#add('MeF0504/dino.vim')
```
or do something like this.

## Usage

The following command will start dino game.
```vim
Dino
```
In the game,

- \<space>: jump
- q: quit
